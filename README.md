# ATM Management System

## Overview
ATM Management System is a Java console-based application that simulates basic ATM operations. Users can authenticate using a PIN, check account balances, and withdraw money securely through a menu-driven interface.

## Features
- PIN-based authentication
- Balance inquiry
- Cash withdrawal
- Multiple account support
- Menu-driven interface
- Input validation
- Object-Oriented Programming implementation

## Technologies Used
- Java
- OOP Concepts
- Scanner Class
- Switch Case Statements

## Concepts Covered
- Classes and Objects
- Encapsulation
- Access Modifiers
- Methods
- Conditional Statements
- User Input Handling
- Switch Statements

## How to Run

### Compile
```bash
javac ATM.java
```

### Execute
```bash
java ATM
```

## Sample Execution

```text
Welcome To The ATM
Please insert your card (a/s/i/j/P/K/C): a

Press 1 to withdraw money
Press 2 to check balance

Enter your PIN: 1234
Your bank balance is 50000
```

## Functionalities

### Check Balance
- User enters a valid card identifier.
- PIN verification is performed.
- Current account balance is displayed.

### Withdraw Money
- User enters a valid card identifier.
- PIN verification is performed.
- Withdrawal amount is entered.
- Remaining balance is displayed after successful withdrawal.
